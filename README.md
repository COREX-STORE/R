# COREX STORE

متجر إلكتروني احترافي لبيع الأجهزة الإلكترونية مع لوحة تحكم كاملة للأدمن

## المميزات

### التصميم
- 🎨 ثيم متحرك بنفسجي وأسود متدرج
- 💧 تأثيرات زجاجية مائية سائلة
- 🌊 خلفية متحركة احترافية
- 📱 تصميم متجاوب (Responsive) لجميع الأجهزة

### قسم الهيدر
- شعار دائري متحرك
- اسم المتجر: COREX STORE
- الوصف والموقع ورقم الهاتف
- روابط التواصل الاجتماعي (TikTok, Instagram, Facebook) مع أيقونات

### قسم المنتجات
- تخطيط عمودين
- صورة المنتج، الاسم، الوصف، السعر
- زر مشاركة المنتج
- عرض الخصم مع السعر الأصلي مشطوب

### لوحة تحكم الأدمن
- تسجيل دخول بحماية كلمة مرور
- قائمة جانبية منزلقة
- إضافة منتجات جديدة
- تعديل وحذف المنتجات
- نظام خصم على الأسعار (نسبة مئوية)
- تبديل الثيمات (4 ثيمات مختلفة)

### التخزين
- ربط بقاعدتين Firebase (نشر متعاقب)
- حفظ كلمة مرور الأدمن
- استرجاع وعرض المنتجات في الوقت الفعلي

## إعداد Firebase

### قاعدة البيانات الأولى (corex-store)
- انتقل إلى Realtime Database
- اذهب إلى Rules
- الصق قواعد `database-rules.json`
- انقر نشر

### قاعدة البيانات الثانية (corex-store-2374d)
- كرر نفس الخطوات السابقة

### قواعد الأمان الموصى بها للإنتاج
```json
{
  "rules": {
    ".read": true,
    ".write": "auth != null || root.child('admin').child('password').exists()",
    "products": {
      ".indexOn": ["timestamp"]
    },
    "admin": {
      ".read": true,
      ".write": "!root.child('admin').child('password').exists() || auth != null"
    }
  }
}
```

## كيفية الاستخدام

### للمستخدم العادي
1. تصفح المنتجات في الصفحة الرئيسية
2. انقر على زر "مشاركة" لمشاركة أي منتج
3. تواصل مع المتجر عبر روابط التواصل الاجتماعي

### للأدمن
1. انقر على أيقونة 👤 أعلى اليسار
2. أدخل كلمة المرور (الأولى تُحفظ تلقائياً)
3. من القائمة الجانبية:
   - إضافة منتج جديد
   - تعديل أو حذف منتج
   - تغيير الثيم
   - تفعيل/إلغاء خصم على المنتج

## رفع صور المنتجات

لرفع صور للمنتجات:
1. اذهب إلى https://catbox.moe
2. ارفع الصورة
3. انسخ الرابط المباشر
4. الصقه في حقل "رابط صورة المنتج"
